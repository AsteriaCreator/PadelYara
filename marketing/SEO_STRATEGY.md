# PadelYara — SEO Strategy

Last updated: 2026-06-06

---

## Why SEO Is Priority

Austrian padel is early. Playtomic is absent. No aggregator exists. No one owns the search results yet.
This is a first-mover window. It closes as the sport grows and competitors notice.

The goal: own Austrian padel search intent before anyone else bothers.

## Technical SEO Priorities

1. **Title tags and meta descriptions** — every page optimized, German-first
2. **Structured data** — LocalBusiness or SportsActivityLocation schema for venue listings
3. **Page speed** — already on Vercel (fast), Lighthouse score should stay green
4. **Mobile-first** — primary user is on their phone making same-day decisions
5. **Sitemap.xml** — submit to Google Search Console immediately
6. **robots.txt** — confirm correct
7. **Canonical URLs** — no duplicate content from search parameters

## Competitive Search Landscape

Current analysis (June 2026):
- No Austrian padel aggregator ranks for these terms
- Individual club websites rank for their own names
- eTennis and Eversports rank for generic booking terms
- Playskan (UK) is not visible in Austrian results

**Opportunity:** The search result page for "Padel Court Wien buchen" is mostly club websites and eTennis. PadelYara can own the aggregator angle immediately with focused content.
